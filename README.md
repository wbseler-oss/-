# Профессиональный интрадей-скринер MOEX

Веб-приложение для дневного трейдинга по российским акциям (TQBR):

- анализ выбранной акции в интрадей,
- график цены + точки **покупки** и **закрытия сделки**,
- сигналы `КУПИТЬ / ПРОДАТЬ/ЗАКРЫТЬ / ОЖИДАТЬ`,
- скринер по большому списку российских акций в таблице,
- автообновление каждые 15 секунд.

## Теханализ (используется в скоринговой модели)

- EMA9 / EMA21
- RSI(14)
- MACD + гистограмма
- Bollinger Bands (20, 2)
- Проверка объема (импульс)

## Запуск

```bash
python3 app.py
```

Открыть:

- UI: `http://127.0.0.1:10000/`
- API по акции: `http://127.0.0.1:10000/api/daytrade/analyze?ticker=SBER`
- API скринера: `http://127.0.0.1:10000/api/daytrade/screener?limit=80`

## Deploy (Render)

- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`

## Важно

Это учебная аналитическая система. Она не дает гарантированных или "точных" прогнозов и не является индивидуальной инвестиционной рекомендацией.
