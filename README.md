# Профессиональный интрадей-скринер MOEX

Приложение для дневного трейдинга по российским акциям (TQBR):

- интрадей-анализ выбранной акции,
- график + точки **Покупка** / **Закрытие**,
- скринер по множеству акций,
- сигналы: `КУПИТЬ / ПРОДАТЬ/ЗАКРЫТЬ / ОЖИДАТЬ`,
- автообновление каждые 15 секунд.

## Улучшенная стратегия

Модель использует несколько фильтров одновременно:

- тренд: EMA9/EMA21/EMA50,
- импульс: MACD histogram,
- состояние рынка: RSI(14),
- справедливая цена: VWAP,
- риск: ATR(14), стоп/тейк/трейлинг,
- волатильность и подтверждение объёмом.

Также считается статистика по сделкам в текущем интрадей-окне:

- `winrate`,
- `profit factor`,
- средняя доходность сделки.

## Запуск

```bash
python3 app.py
```

Открыть:

- UI: `http://127.0.0.1:10000/`
- API анализа: `http://127.0.0.1:10000/api/daytrade/analyze?ticker=SBER`
- API скринера: `http://127.0.0.1:10000/api/daytrade/screener?limit=80`

## Deploy (Render)

- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`

## Важно

На реальном рынке невозможно гарантировать 70-80% прибыльных сделок постоянно.
Это учебная аналитическая система и не является индивидуальной инвестиционной рекомендацией.
