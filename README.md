# Дневной трейдер MOEX (веб + API)

Приложение для интрадей-анализа российских акций (TQBR, Мосбиржа):

- показывает график цены внутри дня,
- отмечает точки **«Покупка»** и **«Закрытие сделки»**,
- выдает текущий сигнал на русском языке,
- позволяет переключаться между акциями из списка MOEX,
- поддерживает автообновление (режим «почти реального времени»).

## Что анализируется

- Источник данных: **MOEX ISS API**.
- Интервал свечей: **10 минут**.
- Логика сигналов (упрощенно):
  - пересечение EMA9/EMA21,
  - фильтр по RSI(14),
  - итог: `BUY` / `SELL` / `HOLD` с русской расшифровкой.

## Запуск локально

```bash
python3 app.py
```

Откройте:

- UI: `http://127.0.0.1:10000/`
- API анализа: `http://127.0.0.1:10000/api/daytrade/analyze?ticker=SBER`
- API списка акций: `http://127.0.0.1:10000/api/tickers`

## Deploy на Render

- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app`

## Важно

- Это учебный инструмент, не индивидуальная инвестиционная рекомендация.
- Перед реальной торговлей обязательно учитывайте риск-менеджмент.
