# Профессиональный интрадей-скринер MOEX

Сервис для дневной торговли российскими акциями (TQBR, Мосбиржа):

- **BUY** = вход в лонг,
- **SELL** = вход в шорт,
- **CLOSE** = закрыть текущую позицию / не открывать новую.

## Что есть в интерфейсе

- Поисковая строка акции (быстрый выбор тикера).
- График цены с точками BUY / SELL / CLOSE.
- Визуальный индикатор силы быков/медведей (зелёная/красная полоска + %).
- Список недавно анализируемых акций (быстрый повторный запуск анализа).
- Отдельная таблица **рекомендуемых** акций (уверенность 80%+) только для сигналов BUY/SELL + режим рынка по каждой акции.
- Клик по строке рекомендации сразу открывает график выбранной акции.

## Стратегия (улучшенная)

Используются совместно:

- EMA9/EMA21/EMA50 (тренд и структура),
- RSI(14),
- MACD histogram,
- VWAP,
- ATR(14),
- ADX(14) + CHOP(14) для фильтра силы тренда и отсечения боковика,
- фильтр волатильности,
- риск-менеджмент: стартовый stop-loss + трендовый trailing stop,
- подтверждение разворота по 3 подряд барам + ослабление ADX перед выходом,
- увеличенное минимальное удержание позиции для захвата крупных движений,
- в боковике сделки не открываются (режим RANGE),
- разделение логики для лонга и шорта,
- в API дополнительно отдается время начала/завершения трендовой фазы (`trend_start_time`, `trend_end_time`),
- возвращается сила рынка: `bull_strength_pct` / `bear_strength_pct` и `market_bias_ru`,
- поддерживаются не только крупные, но и локальные трендовые фазы (`TREND_WEAK`) для более частых сигналов.

Показываются метрики качества в текущем интрадей-окне:

- `winrate`,
- `profit factor`,
- `avg_trade_pct`.

## Запуск

```bash
python3 app.py
```

Открыть:

- UI: `http://127.0.0.1:10000/`
- Анализ акции: `http://127.0.0.1:10000/api/daytrade/analyze?ticker=SBER`
- Рекомендации 80%+: `http://127.0.0.1:10000/api/daytrade/recommended?limit=120&confidence=80`

## Deploy (Render)

- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`

## Важно

Рынок не позволяет гарантировать фиксированный процент прибыльных сделок (например, 70–80%) на постоянной основе.
Это учебная аналитическая система и не является индивидуальной инвестиционной рекомендацией.
