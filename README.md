# Профессиональный интрадей-скринер MOEX

Сервис для дневной торговли российскими акциями (TQBR, Мосбиржа):

- **BUY** = вход в лонг,
- **SELL** = вход в шорт,
- **CLOSE** = закрыть текущую позицию / не открывать новую.

## Что есть в интерфейсе

- Поисковая строка акции (быстрый выбор тикера).
- График цены с точками BUY / SELL / CLOSE.
- Отдельная таблица **рекомендуемых** акций (уверенность 80%+) только для сигналов BUY/SELL.
- Клик по строке рекомендации сразу открывает график выбранной акции.

## Стратегия (улучшенная)

Используются совместно:

- EMA9/EMA21/EMA50 (тренд и структура),
- RSI(14),
- MACD histogram,
- VWAP,
- ATR(14),
- фильтр волатильности,
- риск-менеджмент: stop-loss, take-profit, trailing stop,
- разделение логики для лонга и шорта.

Показываются метрики качества в текущем интрадей-окне:

- `winrate`,
- `profit factor`,
- `avg_trade_pct`.

## Запуск

```bash
python3 app.py
```

Открыть:

- UI: `http://127.0.0.1:10000/`
- Анализ акции: `http://127.0.0.1:10000/api/daytrade/analyze?ticker=SBER`
- Рекомендации 80%+: `http://127.0.0.1:10000/api/daytrade/recommended?limit=120&confidence=80`

## Deploy (Render)

- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`

## Важно

Рынок не позволяет гарантировать фиксированный процент прибыльных сделок (например, 70–80%) на постоянной основе.
Это учебная аналитическая система и не является индивидуальной инвестиционной рекомендацией.
