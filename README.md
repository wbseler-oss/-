# Профессиональный робот-аналитик российских акций

Проект анализирует российские акции (MOEX, доска TQBR) и выдает торговые сигналы:

- `BUY` — при восходящем тренде и приемлемом RSI,
- `SELL` — при нисходящем тренде,
- `HOLD` — если сигналы смешанные.

Доступно 2 режима:

- консольный (`program.py`),
- веб-интерфейс + API (`app.py`, Flask).

## Как это работает

Для каждого тикера программа:

1. Загружает историю цен с MOEX ISS API.
2. Считает индикаторы `SMA` (быстрая и медленная) и `RSI(14)`.
3. Формирует рекомендации `BUY/HOLD/SELL` по таймфреймам.
4. Возвращает агрегированный сигнал через `run_analysis(ticker)`.

## Локальный запуск (консоль)

```bash
python3 program.py
```

## Локальный запуск (веб)

```bash
python3 app.py
```

После запуска откройте:

- Главная страница: `http://127.0.0.1:10000/`
- API: `http://127.0.0.1:10000/api/analyze?ticker=SBER`

## Render

- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app`

## Важно

- Это учебный робот и не является финансовой рекомендацией.
- Перед реальными сделками обязательно проводите дополнительный анализ и учитывайте риск-менеджмент.
